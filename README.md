# BookBook (BB)

## WARNING
Make sure to have Docker running on your System before testing or starting the Application!

## Description
JPA Mapped Model for a LibraryManager

## Table of Contents

- [Usage](#usage)
- [Features](#features)
- [ApiInfo](#ApiInfo)

## Setting up the project

Clone the Repository, open the ProjectFolder (sj24-25-Scherzer-Rabar) with an IDE
(if needed configure ProjectStructure SDK to Amazon Corretto 22.0.2)
When done correctly it should be ready to build & test

## Features

- Basic Model
- Postgres Database with local temporary Container & testdata
- Services
- Projections
- Controller
- Documentation
- Test to all of the Features mentioned Above!

## ApiInfo

For the API start the Application via TestMainClass and enter "localhost:8080" in the browser, further Information About the Documentation will be provided there!
